Spd-University - про нас

Що собою являє SPD University?

SPD-University - це проект, що допомагає талановитій молоді, яка хоче пов'язати своє життя з ІТ, максимально нівелювати той самий розрив між теоретичними знаннями та досвідом, необхідним для працевлаштування.

Програми SPD-University розроблені спеціалістами, що працюють в ІТ, а відтак найбільше відповідають очікуванням роботодавців. Розроблюючи програми навчання, провідні фахівці орієнтуються на те, що частина слухачів курсів буде працювати в їхній команді, а отже максимально адаптують їх під робоче середовище.

Усі програми мають "університетський" підхід. Це означає, що крім основних дисциплін зі спеціалізації у слухачів також будуть лекції та практичні заняття із суміжних дисциплін. Одним з найважливіших елементів навчання є наскрізний проект, який виконується в повноцінних командах, що складається з Back-End і Front-End розробників, а також QA. Менторами проектів виступають провідні спеціалісти з компанії SPD-Ukraine.

Команда SPD складається з найдосвідченіших спеціалістів своєї області, що готові ділитися своїм практичним досвідом. Курси безкоштовні, проте вимагають від абітурієнтів теоретичної бази.

На данний момент у нас відкриті такі курси:

  • Java for Web
  • Front-End
  • Quality Assurance

Курси SPD-University

Java for Web

Курс розрахований на тих, хто вже має теоретичну базу та зробив свої перші кроки у вивченні цієї ентерпрайз технології

Front-End

Цей курс призначений для тих, хто хоче навчитися створювати сайти та динамічні веб-додатки

QA

Курс призначений для слухачів без професійного досвіду в сфері тестування програмного забезпечення

Як потрапити на курси?

Що робити, щоб потрапити на курси в SPD-University?

  • Зареєструватися на нашому сайті або за посиланням у FB
  • Пройти тестування з напрямку, який ви обрали під час реєстрації
  • У разі успішного складання тесту - пройти співбесіду

Деталі про вступ до SPD-University

Для того, щоб потрапити до SPD-University потрібно:

Крок перший - зареєструватися на курси на офіційному сайті SPD-University або на сторінці у Facebook. Вам обов‘язково прийде підтвердження про успішну реєстрацію на пошту.

Другий крок - складання авто-тесту з обраного вами напряму (англійською мовою). Найкращих за результатами кандидатів ми запросимо на співбесіду.

Важливо! Для напрямку QA тестів буде 2. Один профільний, другий - на визначення рівня англійської мови.

Третій і останній крок - співбесіда. У вас буде можливість продемонструвати технічні знання та комунікативні навички.

Вимоги до кандидатів

Java for Web

  • Java 8 Core: класи/інтерфейси, Generic, Collections API, IO, Exceptions, аннотації.
  • Принципи ООП.
  • Знання алгоритмів і структур данних.
  • Основи SQL.
  • Основи HTML/CSS/JS.
  • Основи Web: HTTP запити, Cookies, Session.
  • Рівень англійської Intermediate або вище.

Front-end

  • Знання основ HTML/CSS.
  • Вміння користуватися одним із графічних редакторів.
  • Досвід верстки від 3 місяців.
  • Досвід Responsive або Adaptive верстки
  • Розуміння основ програмування, структур данних та алгоритмів.
  • Базові знання Javascript.
  • Рівень англійської Intermediate або вище.

QA

  • Рівень англійської Intermediate або вище.
  • Базова комп'ютерна грамотність.
